# Vue Storyblok Boilerplate

This is a boilerplate to make a headless site with storyblok

## Project Setup

```sh
cp env.example > .env

// add correct credentials

npm ci
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Fetch component types

```sh
npm run storyblok
```

## Requirements

- Storyblok CLI
- Node v22+
- NPM v10+
